# Task 5: Logistic Regression - Breast Cancer Dataset

## Objective
Build a Logistic Regression model for binary classification using the Breast Cancer dataset.

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Jupyter Notebook / Google Colab

## Dataset
Breast Cancer Dataset from Scikit-learn

## Steps Performed
1. Import required libraries
2. Load Breast Cancer dataset
3. Split data into training and testing sets
4. Apply feature scaling using StandardScaler
5. Train Logistic Regression model
6. Predict classes on test data
7. Evaluate using:
   - Accuracy Score
   - Confusion Matrix
   - Classification Report

## Expected Result
Model achieves high accuracy and helps understand binary classification using Logistic Regression.
